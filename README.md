# git-practice

本リポジトリは、P2HACKS2025 勉強会～実践編～の参加者が、演習を行うためのリポジトリです。

本リポジトリは、参加者が次のステップを踏むことで、Gitの基本操作を理解することを目指します:

- 本リポジトリをFork
- Fork先をPull
- ファイルを追加
- ファイルをCommit
- ファイルをPush
- Pushした内容でPull Requestを作成

参加者は、本リポジトリをForkし、次の内容をCommitしてPull Requestを出してださい:

- 自分のプロフィールに関するファイル（`名前.txt`）
  - ファイル名は、他人と被らず自分だとわかるものにしてください
    - 本名は推奨しません
    - ハンドルネームがあれば最適です
  - ファイルの中身には、以下を記述してください
    - 学年（例: B1, B2, etc...）
    - ハッカソン参加歴（例: 初めて, 3回目, etc...）

ファイルの例は`azurata.txt`に記載しています。
